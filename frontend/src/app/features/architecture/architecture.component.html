Routing

Lazy, feature-owned routes

The root route mounts a single MainLayoutComponent from core/layout. Each feature is lazy loaded with loadComponent, aligning the URL tree to the folder tree. 

app/
 ├─ core/
 │   └─ layout/main-layout.component
 ├─ features/
 │   ├─ home/home.component
 │   └─ architecture/architecture.component
 └─ shared/
